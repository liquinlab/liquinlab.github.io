# liquinlab_website

Instructions for making edits to the website:

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Reformat

```sh
npm run format
```

### Compile and Minify for Production

```sh
npm run build
```

### Deploy to live website

```sh
npm run deploy
```

After build and deploy, commit and push changes to GitHub.
